# 소개
DYB팀이 개발한 장애인 주차구역 단속 시스템 프로젝트입니다.

## 프로젝트 개요
장애인 주차구역의 불법 주차 문제를 보다 효율적으로 단속할 수 있도록 기획 및 개발한 프로젝트입니다.  
기존 단속 방식의 불편함을 줄이고, 실제 현장에서 활용 가능한 단속 환경을 제공하는 것을 목표로 했습니다.

## 주요 특징
- 카메라 모듈 1대를 활용한 단속 시스템 구현
- 기존 거리 측정 모듈 중심 방식과 차별화된 접근
- 현장 활용성을 고려한 시스템 구성
